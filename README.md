---
## 프로젝트 구성
- 데이터 수집: 다양한 환경에서 차량 번호판 이미지를 수집.
- 데이터 전처리: 이미지 정제, 증강 및 라벨링.
- 모델 개발: YOLOv10s 모델을 사용하여 번호판 인식.
- 서비스 구축: 번호판 인식 API 개발 및 사용자 인터페이스 구현.
